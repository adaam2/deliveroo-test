# Starting out

1. `git clone git@github.com:adaam2/deliveroo-test.git`
2.  `cd deliveroo-test`
3. `yarn install`
4. `yarn start`

# UI library

To run Storybook.js:

```
yarn run storybook
```

# Tests

To run tests:

```
yarn test
```

# Tech

- TypeScript
- React
- Storybook
- CSS Modules
- CSS Grid + Flexbox
- React AXE (for accessibillity)
- Jest
- Enzyme

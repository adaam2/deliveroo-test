# Starting out

1. `yarn install`
2. `yarn start`

# UI library

To run Storybook.js:

```
yarn run storybook
```

# Tests

To run tests:

```
yarn test
```

# Tech

- TypeScript
- React
- Storybook
- CSS Modules
- CSS Grid + Flexbox
- React AXE (for accessibillity)
- Jest
- Enzyme
